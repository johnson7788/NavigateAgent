# 后端主程序入口

> 本项目为NavigateAgent系统的后端主API程序

## 简介

本目录包含后端主程序的源代码，提供核心的API服务功能。

## 文件说明

| 文件 | 说明 |
|------|------|
| `main.py` | 主程序入口文件 |
| `requirements.txt` | Python依赖包列表 |
| `test_main.py` | 测试文件 |
| `.env` | 环境变量配置 |

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

## 配置

请在 `.env` 文件中配置必要的环境变量。
